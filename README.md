# 2Selai & a Butter
A Bootstrap redesign of a real South African kota restaurant's website. This is my individual capstone project.

## Theme

A redesign of 2Selai & a Butter, a proudly township-born kota restaurant based
in Protea North, Soweto. The site introduces the business, showcases an
8-item kota menu, tells the "Why Choose Us / Our Mission / What We Do" story,
and gives visitors everything they need to get in touch or find the
restaurant on the map.

## Features

- Single-page layout with Home, Menu, About, and Contact sections linked by
  anchor navigation (`#Home`, `#Menu`, `#About`, `#Contact`)
- Bootstrap 5 grid system throughout (`container`, `row`, `col-lg-*`) layered
  with the "Canvas" theme framework for typography and navbar behaviour
- Responsive 8-item menu grid using Bootstrap's `card` component, with all
  product photos forced to a consistent size using `object-fit-cover` so
  differently-sized original images don't distort the layout
- Custom branding in the navbar logo: an image plus the "2" in 2Selai styled
  red to match the business's identity
- Contact form with a "Type of Enquiry" dropdown built specifically for a
  restaurant (Orders, Event Catering, Franchise/Business Opportunity,
  Feedback or Compliment)
- Embedded Google Map using the restaurant's real address, sitting
  side-by-side with the contact details on desktop and stacking on mobile
- Validated with the W3C HTML validator
